# Bali Real Estate Dominance Dashboard

Минимальный веб‑дашборд на Node.js + GA4 Data API. Показывает трафик, лиды, вовлеченность, охват и индекс доминирования.

## Запуск

1. Создайте `.env` на основе `.env.example`.
2. Укажите `GA4_PROPERTY_ID` и путь к JSON‑ключу сервисного аккаунта.
3. Запустите:

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Лиды

`LEAD_EVENTS` — список событий GA4, которые считаем лидами (через запятую).

## Индекс доминирования

Сравнение текущего периода с предыдущим той же длины.
Весовые коэффициенты:
- Трафик: 35%
- Лиды: 35%
- Вовлеченность: 20%
- Охват: 10%

Индекс = 100 * взвешенная сумма коэффициентов роста.
